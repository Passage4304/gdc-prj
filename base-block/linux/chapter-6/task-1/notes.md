### Процессы
```bash
ps -C <command> # поиск процесса по имени команды
ps -e -o user,pcpu,cmd --sort -pcpu # сортировка процессов по самому высокому потреблению cpu
ps -e -o user,pmem,cmd --sort -pmem # сортировка процессов по самому высокому потреблению ram
pstree <user> -p # вывод деревьев процессов пользователя user с отображением PIDs
```
